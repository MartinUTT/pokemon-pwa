# Pokemon PWA

## Descripción

Aplicación Web Progresiva que consume la API pública PokeAPI.

## Tecnologías

- React
- Vite
- TypeScript
- PokeAPI
- Service Worker

## Funcionalidades

- Búsqueda de Pokémon
- Cache offline básico
- Instalación en dispositivo
- Persistencia con LocalStorage

## Cómo instalar

1. Abrir la app en navegador compatible
2. Seleccionar "Agregar a pantalla de inicio"
